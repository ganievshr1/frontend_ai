# 🤖 GigaChat Client - Чат с нейросетью от Сбера

Веб-приложение для взаимодействия с нейросетью GigaChat от Сбера. Поддерживает потоковую генерацию, Markdown-форматирование, работу с изображениями и управление чатами.

## 📋 Функциональность

### Основные возможности
- ✅ **Чат с нейросетью GigaChat** — полноценный диалог с ИИ
- ✅ **Потоковая генерация ответов (streaming)** — ответы появляются токен за токеном
- ✅ **Поддержка Markdown** — заголовки, списки, код, ссылки, таблицы
- ✅ **Подсветка синтаксиса кода** — более 100 языков программирования
- ✅ **Отправка изображений (multimodal)** — анализ и описание изображений
- ✅ **Копирование ответов** — один клик для копирования в буфер обмена
- ✅ **Остановка генерации** — прерывание длинных ответов

### Управление чатами
- ✅ **Создание чатов** — автоматическая генерация названия из первого сообщения
- ✅ **Переименование чатов** — редактирование названия
- ✅ **Удаление чатов** — с подтверждением
- ✅ **Поиск по чатам** — по названию и содержимому
- ✅ **Сохранение истории** — в localStorage (не теряется после перезагрузки)

### Настройки
- ✅ **Temperature** (0-2) — креативность ответов
- ✅ **Top-P** (0-1) — разнообразие ответов
- ✅ **Max Tokens** — максимальная длина ответа
- ✅ **System Prompt** — системный промпт
- ✅ **Темная/светлая тема** — для комфортной работы

### Пример 3: Написание кода
👤 Пользователь: Напиши функцию для сортировки массива объектов по полю "age"

🤖 GigaChat: Вот функция для сортировки:

javascript
/**
 * Сортирует массив объектов по указанному полю
 * @param {Array} arr - Исходный массив
 * @param {string} field - Поле для сортировки
 * @param {boolean} ascending - Порядок сортировки (true - по возрастанию)
 * @returns {Array} Отсортированный массив
 */
function sortByField(arr, field, ascending = true) {
  return [...arr].sort((a, b) => {
    if (a[field] < b[field]) return ascending ? -1 : 1;
    if (a[field] > b[field]) return ascending ? 1 : -1;
    return 0;
  });
}

// Пример использования
const users = [
  { name: 'Анна', age: 25 },
  { name: 'Борис', age: 30 },
  { name: 'Виктор', age: 22 }
];

const sortedByAge = sortByField(users, 'age', true);
console.log(sortedByAge);
// Результат: Виктор(22), Анна(25), Борис(30)

## 🐛 Устранение неполадок

### Ошибка "Failed to fetch"

**Проблема:** Фронтенд не может подключиться к бэкенду

**Решение:**
1. Убедитесь, что бэкенд запущен:
   ```bash
   cd backend_for-_rontend
   npm run dev
Проверьте, что порт 3001 не занят

Проверьте CORS настройки в server.js

Ошибка авторизации (401 Unauthorized)
Проблема: Неверный Authorization Key

Решение:

Проверьте, что ключ скопирован полностью (без лишних пробелов)

Убедитесь, что ключ актуален (срок действия 6 месяцев)

Получите новый ключ в GigaChat Studio

Проверьте, что вы выбрали правильный Scope

Изображение не отправляется
Проблема: Ошибка при отправке изображения

Решение:

Убедитесь, что изображение не превышает 5MB

Поддерживаемые форматы: JPEG, PNG, GIF, WebP

Проверьте, что в запросе правильно формируется base64

Убедитесь, что модель поддерживает мультимодальность (GigaChat, GigaChat-Pro)

Медленные ответы
Проблема: API долго отвечает

Решение:

Используйте streaming режим (ответы появляются постепенно)

Уменьшите max_tokens для коротких ответов

Выберите модель GigaChat-Plus для скорости

Проверьте качество интернет-соединения

Потеря истории чатов
Проблема: После перезагрузки страницы чаты исчезли

Решение:

Проверьте, что localStorage не очищен браузером

Убедитесь, что в настройках браузера разрешены куки

Используйте ручной экспорт/импорт через кнопку "Сохранить настройки"
